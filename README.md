# WebRTC-Live-Stream
## usage
Require Node.js
1. `cd WebRTC-Live-Stream/`
2. `npm install`
3. `npm start`
4. Server running on http://localhost:3000

## note
If client using chrome
1. open chrome://flags/#unsafely-treat-insecure-origin-as-secure
2. add server's address in textarea below
3. choose "Enable"
